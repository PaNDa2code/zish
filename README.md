# zish

opinionated shell for personal use. targets full bash/sh compatibility.

## Features

- vim-style editing modes
- tab completion
- command history
- configurable aliases (~/.zishrc)
- colorized output

## Build

```
zig build
./zig-out/bin/zish
```
